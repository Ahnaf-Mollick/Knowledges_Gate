# Knowledges Gate 📚

A modern Flutter application focused on providing a clean and interactive user experience with secure authentication.  
The project implements **Firebase Email & Password Authentication**, **GetX State Management**, and a beautiful responsive UI design.

---

## 🚀 Features

- 🔐 **Firebase Authentication**
  - Email & Password Sign Up
  - Email & Password Login
  - Secure user authentication flow
  - Firebase user session handling

- ⚡ **GetX State Management**
  - Reactive state management
  - Efficient UI updates
  - Clean controller-based architecture
  - Simple navigation management

- 🎨 **Modern UI Design**
  - Beautiful and clean interface
  - Responsive layouts
  - User-friendly authentication screens
  - Smooth user experience

- 🏗️ **Clean Project Structure**
  - Separation of UI, controllers, and services
  - Maintainable and scalable codebase
